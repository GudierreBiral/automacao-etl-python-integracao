Visão Geral

Esta pasta contém os scripts SQL desenvolvidos para tratamento, consolidação e disponibilização dos dados utilizados no Power BI.

O foco principal foi garantir:
	Consistência dos dados
	Padronização de tipos
	Evitar duplicidades
	Facilidade de consumo analítico

Estrutura
	Procedures de carga e tratamento de dados
	View consolidada para consumo no Power BI
	Scripts de validação de dados

Observações Importantes
	Durante o desenvolvimento, foram identificadas inconsistências na base original, especialmente relacionadas a:
		Volume Real
		NS Real
		Dados de pausas específicas (NR17)
	***Essas limitações foram tratadas por meio de regras de negócio e documentadas para transparência.
